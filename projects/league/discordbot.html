League of Legends Discord Bot

Otherwise known as League of Legends Bint Bot

Binting is when someone ints (intentionally feeds) to the point you consider the possibility they may be part bonobo. [Citation not needed]

Disclaimer

Requests are limited to NA1 servers only and API requests can only occur 100 times every two minutes, meaning using the prediction functionality more than once every two minutes will easily surpass this limit and an error will occur. Additionally, there are certain methods in place where the number of requests depends on the player's ingame habits and the limit can once again be exceeded. Also I need to enter in a new API key every 24 hours so it might also not work because of that.

More information

This projects began when I became interested in learning how to use APIs and League of Legends is a game I am quite familiar with. I explored a lot of funcitonality and wrote many scripts that showcased interesting stats that weren't really found elsewhere on a Jupyter Notebook. The original notebook kinda stopped working so the best I could find is this copy I wanted to have a more accessible and public way of showcasing this so I eventually added the functionality to a Discord Bot so my friends and acquaintances could use it too.

All of the API's were found from https://developer.riotgames.com/apis or the League of Legends ddragon. Python modules requests and discord were used.
