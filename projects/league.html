League of Legends Projects

These projects were mainly done to learn Python, especially the requests library and APIs with a platform I was familiar with.

Basic Information

These projects began when I became interested in learning how to use APIs and League of Legends is a game I am quite familiar with. I explored a lot of funcitonality and wrote many scripts that showcased itneresting stats that weren't really found elsewhere on a Jupyter Notebook and eventually added the functionality to a Discord Bot so my friends and acquaintances could use it too.

The prediction projects was originally a project completed with dungwoong, a man with big data science brain, and was later added to the bot so that others could use it too.

Disclaimer about the bot

Requests are limited to NA1 servers only and API requests can only occur 100 times every two minutes, meaning using the prediction functionality more than once every two minutes will easily surpass this limit and an error will occur. Additionally, there are certain methods in place where the number of requests depends on the player's ingame habits and the limit can once again be exceeded. Also I need to enter in a new API key every 24 hours so it might also not work because of that.
